# 《从 Prompt 到 Harness：AI Agent 的工程化方法论》

这是一本关于 Agent 时代软件工程如何被重写的方法论书稿，讨论的不是“怎样多用一点 AI”，而是“当 AI 真正进入工作后，工程、组织与管理要怎样一起重写”。
